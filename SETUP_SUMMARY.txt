TROUBLESHOOTING QUICK REFERENCE
================================================================================

Issue: "BLOB_READ_WRITE_TOKEN not set"
Solution: export BLOB_READ_WRITE_TOKEN="vercel_blob_rw_..."

Issue: "ImageMagick not found" or "Pillow not found"
Solution: pip install Pillow

Issue: Download fails for file ID
Solution: Verify file ID is correct, try manual download

Issue: Upload to Blob fails
Solution: Check token, verify WebP files exist, check connection

Issue: Generated URLs return 404
Solution: Wait 5 minutes for CDN, check path is correct

Issue: Script hangs
Solution: Check internet, may need to run in smaller batches

See IMAGES_UPLOAD_README.md or VILLA_IMAGES_SETUP.md for full troubleshooting
